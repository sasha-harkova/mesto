# Project: Mesto

This is a social networking site where **you can post photos, like them, delete them and edit your profile**.

Here were used:
* JavaScript;
* OOP;
* Webpack
* grid layout;
* flexbox layout;
* adaptive layout;
* positioning properties;
* BEM technology;
etc
